Odoo Debranding

Description

Odoo Debranding Module is designed to remove all visible traces of Odoo branding throughout your Odoo instance, giving your application a white-label appearance.

This module allows you to customize the platform with your own logos and branding, ensuring that no references to Odoo remain on any visible screens. Perfect for businesses wanting a fully branded solution, this module enables you to present a customized, professional interface to users and clients.

Features

  • → Complete Debranding
  • → Custom Logo Upload
  • → Brand Customization
  • → Footer Customization
  • → Eliminates "OdooBot" in all server logs.
  • → Login Screen Modification
  • → Email Debranding
  • → No "Powered by Odoo" Labels
